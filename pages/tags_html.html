Tags em HTML

em HTML5, podemos inserir os elementos da lista a seguir. passe o mouse sobre os termos para descobrir qual tag devera utilizar em cada caso

nem todas as tags precisam ser fechadas, apenas as sequidas de '</>' precisam ser fechadas

Tags de texto

Titulos

use - <h1> </h1> - cria titulos de primeiro nivel, vai do h1 ate o h6.

Paragrafo

use - <p> </p> - cria paragrafos.

Abreviações

use - <abbr title=""> </abbr> - "title" recebe o valor completo do texto sendo abreviado.

Codigos-fonte

use - <code> </code> - isso faz o texto usar uma fonte mono-espaçada.

Listas numeradas

use - <ol> </ol> - "Ordered List".

Listas demarcadas

use - <ul> </ul> - "Unordered List".

Item da lista

use - <li> </li> - "List Item" (não precisa ser fechada).

Negrito

use - <strong> </strong> cria um texto forte.

Italico

use - <em> </em> - cria um texto com enfase.

Link (a)

use - <a href="" target> </a> - "href" recebe o caminho a ser linkado, pode ser um url ou outra pagina do seu site(outro arquivo html). Target recebe o valor que diz se o link abrira em outra aba do navegador ou na propria aba, sendo os valores respectivamente: "_self" para abrir na propria pagina, e "_blank" para abrir em uma nova aba.

Tags de midias

Atente-se: apenas as tags "video" e "audio" precisam ser fechadas.

img

use - <img src="" alt=""> - src recebe o caminho da imagem, alt recebe um texto descritivo para o caso da imagem nao ser mostrada.

picture

use - <source media="(max-width: 750px)" srcset="img/img_p.png" type="image/png">

audio

use - <audio src="" controls></audio> - src recebe o caminho da imagem, alt recebe um texto descritivo para o caso da imagem nao ser mostrada. recomenda-se o uso acompanhado da tag "source".

video

use - <video controls> - . recomenda-se o uso acompanhado da tag "source".

Tags conteiners

Atente-se: todas a tags conteiners precisam ser fechadas.

header

Usado apra fazer cabeçalhos.

nav

Feito para criar barras de navegação.

main

Leva todo o conteudo principal do site. Rescomenda-se o uso header > main > footer.

section

Usado para separar seções no site.

article

Artigos.
